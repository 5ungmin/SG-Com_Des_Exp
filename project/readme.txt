'''
사용자가 플레이할 수 있는 Maze Program
'''

[FILE] 버튼을 클릭해 .maz 확장자의 미로 파일을 불러옵니다.
성공적으로 파일을 불러온 경우 화면에 미로가 나타납니다.

[DFS] 버튼을 누르면 DFS 알고리즘을 통해 찾은 경로가 나타납니다.

[BFS] 버튼을 누르면 BFS 알고리즘을 통해 찾은 경로가 나타납니다.

* 주의) DFS 버튼과 BFS 버튼은 동시에 누를 수 없습니다.

[PLAY] 버튼을 누르면 사용자가 방향키를 입력해 탈출 경로를 찾아갈 수 있습니다.
타이머가 작동되니 최단 시간에 탈출해보세요!

[QUIT] 버튼을 누르거나 키보드에서 'q'를 입력하면 프로그램이 종료됩니다.
(Xcode에서 실행 시, 프로그램 실행 전 키보드 언어 설정이 *영어*로 되어있어야 q 키가 작동합니다.)
